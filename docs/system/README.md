## 📚 ドキュメント一覧

このフォルダには、KeibaAI_v2システムの詳細な技術ドキュメントが含まれています。

### 推奨読書順序

初めての方は、以下の順序で読むことをお勧めします：

1. **[01_システム概要.md](./01_システム概要.md)**
   - プロジェクトの目的、全体像、技術スタック
   - **対象**: 全員（必読）
   - **所要時間**: 15分

2. **[02_アーキテクチャ.md](./02_アーキテクチャ.md)**
   - システムアーキテクチャ、ディレクトリ構造、モジュール設計
   - **対象**: 開発者、システム設計者
   - **所要時間**: 20分

3. **[03_データモデル.md](./03_データモデル.md)**
   - Parquetスキーマ、主キー・外部キー、データ型定義
   - **対象**: データエンジニア、開発者
   - **所要時間**: 25分

4. **[04_スクレイピング詳細.md](./04_スクレイピング詳細.md)**
   - Webスクレイピングの実装、BAN対策、エラーハンドリング
   - **対象**: スクレイピング担当者、開発者
   - **所要時間**: 20分

5. **[05_パース処理詳細.md](./05_パース処理詳細.md)**
   - HTML解析パターン、4種類のパーサー、共通ユーティリティ
   - **対象**: パーサー開発者
   - **所要時間**: 30分

6. **[06_特徴量エンジニアリング.md](./06_特徴量エンジニアリング.md)**
   - 6つの特徴量カテゴリ、FeatureEngineクラス、欠損値処理
   - **対象**: データサイエンティスト、ML開発者
   - **所要時間**: 30分

7. **[07_機械学習モデル.md](./07_機械学習モデル.md)**
   - 3パラメータモデル（μ, σ, ν）、LightGBM設定、確率校正
   - **対象**: ML開発者、データサイエンティスト
   - **所要時間**: 35分

8. **[08_シミュレーションと最適化.md](./08_シミュレーションと最適化.md)**
   - モンテカルロシミュレーション、Kelly基準、ポートフォリオ最適化
   - **対象**: クオンツ開発者、ML開発者
   - **所要時間**: 25分

9. **[09_設定とテスト.md](./09_設定とテスト.md)**
   - YAML設定ファイル、テスト戦略、デバッグツール
   - **対象**: 開発者、QAエンジニア
   - **所要時間**: 20分

10. **[10_運用ガイド.md](./10_運用ガイド.md)**
    - セットアップ、日次運用フロー、トラブルシューティング
    - **対象**: 運用担当者、全員
    - **所要時間**: 25分
